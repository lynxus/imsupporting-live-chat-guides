## Installing the IMsupporting Live Chat Widget

Typical installation steps:

1.  Log into your dashboard
2.  Navigate to Website Ontegrations
3.  Copy your widget code
4.  Insert the script before the closing
    ```{=html}
    </body>
    ```
    tag

Example:

``` html
<script src="https://widget.chatcdn.imsupporting.com/widget.js"></script>
```
